# xDPU – Cross-Domain Dedicated Processing Unit

**A no-moving-parts, optically-interconnected, 3D‑stacked heterogeneous processor for extreme power efficiency.**

## Abstract

The xDPU integrates a slim CPU, parallelization unit (PPU), central math unit (MPU), neural engine (NPU), and GPU into a single 3D‑stacked package. Shared cache, on‑chip silicon photonics, and thermal interleaving eliminate redundant data movement and active cooling – enabling 65‑80% lower power than conventional SoCs, with no fans or pumps. Ideal for space, edge, and mobile.
